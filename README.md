# 📊 Customer Churn Analysis Dashboard (Power BI)

## 📌 Project Overview

This project presents an interactive Power BI dashboard designed to analyze telecom customer churn behavior. The goal is to identify high-risk customer segments and support data-driven retention strategies using dynamic DAX calculations and executive-level reporting techniques.

The dashboard simulates a real-world business intelligence scenario where management needs clear KPIs, risk classification, and actionable insights.

---

## 🚀 Key Features

- 📈 Dynamic Churn Rate Calculation
- 🔴🟡🟢 Risk-Level Classification (High / Medium / Low)
- 🎨 Conditional Formatting using DAX
- 🧠 Executive-Level Dynamic Insight Card
- 🎛 Interactive Slicers (Contract Type, Internet Service, Payment Method, etc.)
- 📊 KPI Cards for Customer Overview

---

## 🛠 Tools & Technologies Used

- Power BI Desktop
- DAX (Data Analysis Expressions)
- Data Modeling
- Conditional Formatting Logic
- Git & GitHub (Version Control)
